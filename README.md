# Code with Claude 2026 | London Conference Index

## Meta Key Insights
*Overarching strategic themes synthesized from the Code with Claude London 2026 sessions.*

- **Agentic Orchestration & The "Intent" Shift**: Moving from manual coding to orchestrating "agent armies" where human value lies in defining intent and performing high-level architectural review.
- **The Verification Paradigm (Grader-in-Chief)**: As production volume explodes, the primary engineering skill shifts from "creation" to "verification"—using rubrics (Outcomes), evals, and automated tests to steer agents.
- **Rigorous Context Hygiene**: Treating the context window as a "fixed box" and using tools like `CLAUDE.md`, XML tags, and prefix-aware prompt caching to maximize agent performance and cost-efficiency.
- **Asynchronous Engineering & Routines**: Transitioning from reactive, synchronous prompting to "always-on" agents that perform maintenance, bug triage, and refactoring asynchronously via webhooks or schedules.
- **AI-Native Organizational Design**: Rebuilding teams around the "AI-native engineer"—managers-as-ICs, hub-and-spoke pods, and hypergrowth velocity enabled by agentic onboarding.
- **Production-Ready AI Primitives**: Leveraging Managed Agents, self-hosted sandboxes, and MCP (Model Context Protocol) to bridge the gap between local prototypes and secure enterprise deployments.
- **Economic Efficiency & The Advisor Strategy**: Shifting the efficiency frontier using Prompt Caching (90% savings) and the "Advisor Strategy" (using Opus to guide smaller models).
- **Domain-Specific Agent Inheritance**: Applying the "structural DNA" of coding agents (Read-Edit-Verify loops, sandboxing, linters) to non-technical verticals like Law, Health, and Logistics.
